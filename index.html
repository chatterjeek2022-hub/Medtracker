Save all medicines, purchases, doses and records as a JSON file. Store it in Google Drive, iCloud Drive or any folder.

Restore data from a previously exported file. Works on any device — Android or iPhone.

📱
Switching phones?
On old phone — tap Export and save the file to Google Drive or iCloud Drive.
Install this HTML file on your new phone (open in browser → Add to Home Screen).
On new phone — tap Import and pick the file from your cloud drive.
All your data is restored instantly.
💡 The app works identically on Android (Chrome) and iPhone (Safari). Same file, same data format.
